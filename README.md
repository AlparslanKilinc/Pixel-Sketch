
# Etch-a-Sketch
#### Sketching Website
You are able to click and drag to draw on the grid, giving you full ability to draw as you please.

##### Options
* Grid Size
* Color Pallete
* Random Color Generator 
* Eraser
* Border Lines On/Off
* Clear

Live Preview:  https://alparslankilinc.github.io/Etch-a-Sketch/
